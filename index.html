PyGPTPrompt

description

PyGPTPrompt is a CLI tool that manages context windows for AI models, facilitating user interaction and data ingestion for optimized long-term memory and task automation. While it was initially designed for the OpenAI REST API, it now supports a wider range of AI models.

* This application requires an API key from the respective AI model provider.
